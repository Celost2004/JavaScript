
# Variaveis

## Declaração

let -> valor pode ser reatribuido ou modificado ao longo do tempo
var -> antiga forma de declara variáveis semelhante a let
const -> valores imutaveis

## Tipos

string -> dados formato caracteres
int ->  dados do tipo numericos inteiros.
array -> conjuntos de dados podendo conte diversos tipos de dados
bool -> dados boleano true (verdadeiro) e  false (falso)



# Operadores

## Aritimeticos
+ somar
- subtrair
* multiplicar
/ dividir
% resto da divisão

## Comparadores Matematicos
< menor que
> maior que
<= menor ou igual
>= maior ou igual

## Comparadores Lógicos

== igualdade entre sentenças (valor)
!= diferença entre sentenças (valor)
=== igualdade entre sentenças (valor e tipo)
!== diferença entre sentenças (valor e tipo)

## Operadores de lógica e junção lógica

! NÃO (NOT)
&& E (AND)
|| OU (OR)

O sinal de exclamação (!) é o operador NOT (não), utilizado para negar a sentença que vem na sequencia.

#### As condições lógicas são convertidas em números binários:
true é equivalente a 1
false é equivalente a 0

#### Operador lógico de atribuição

Tem a capacidade de atribuir o valor de uma variavel a partir de uma condição lógica
